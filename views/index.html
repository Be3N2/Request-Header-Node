Request Header Parser

Simple web api that returns data on the user calling the get request.

To use add /whoami to this page's url for the request and you
will recieve a simple json file that looks something like this:

{"ipaddress": "127.0.0.1", "language": "en-US", "software": "Windows NT 10.0; Win64; x64"}


Here is the general call:

https://basicuserdata.glitch.me/whoami
